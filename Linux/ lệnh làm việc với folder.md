## Các lệnh làm việc với thư mục
### lệnh -cd
 + chuyển đến thư mục chỉ đinh
###vd :
```
   root@chungpt:~# cd /etc/
   root@chungpt:/etc#
```
 + trở về thư mục root
```
   root@chungpt:~# cd /etc/
   root@chungpt:/etc# cd 
   root@chungpt:
```
 + trở về thư mục cha
```
   root@chungpt:~#cd /etc/apache2/
   root@chungpt:/etc/apache2/# cd ..
   root@chungpt:/etc# 
```
### Lệnh ls
 + xem liệt kê thư mục
